Personal Portafolio - The Odin Project

This project is part of The Odin Project curriculum, developed to create a responsive personal portfolio using HTML, CSS, and JavaScript.

The goal was to apply web design fundamentals, semantic structure, and interactive visual effects to showcase my skills, projects, and professional networks.

--Project Objectives

Build a modern, static website using HTML, CSS, and JavaScript.

Practice responsive design that adapts to different screen sizes.

Implement a clean and attractive visual design using Flexbox and Grid layouts.

Integrate animations and visual effects (hover, transitions, shadows, etc.).

Display personal information, projects, and links to professional networks.

--Technologies Used
 
HTML5  
CSS3  
JavaScript (ES6)
